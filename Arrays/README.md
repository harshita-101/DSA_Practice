# Arrays

This folder contains my solutions to Array problems for Data Structures and Algorithms (DSA) interview preparation.

## Goal

* Build strong problem-solving skills.
* Understand Brute Force, Better, and Optimal approaches.
* Learn time and space complexity analysis.
* Practice company-level interview questions.
* Write clean and optimized C++ solutions.

## Problems Solved

| S.No. | Problem                              | Difficulty | Status |
| ----: | -------------------------------------| ---------- | --- |
|     1 | Largest Element in an Array          | Easy       | ✅ |
|     2 | Second Largest Element in an Array   | Easy       | ✅ |
|     3 | Check if Array is sorted             | Easy       | ✅ |
|     4 | Remove Duplicates from Sorted Array  | Easy       | ✅ |
|     5 | Left Rotate Array by One Place       | Easy       | ✅ |
|     6 | Left Rotate Array by D Places        | Easy       | ✅ |
