# Largest Element in an Array

## Problem Statement

Given an integer array, find and return the largest element present in the array.

### Example 1

**Input:**

```text
[2, 5, 1, 3, 0]
```

**Output:**

```text
5
```

### Example 2

**Input:**

```text
[8, 10, 5, 7, 9]
```

**Output:**

```text
10
```

---

## Approaches

### 1. Brute Force

* Find the largest element using a basic approach.
* Time Complexity: **O(n)**
* Space Complexity: **O(1)**

### 2. Optimal Approach

* Traverse the array once while keeping track of the largest element found so far.
* Time Complexity: **O(n)**
* Space Complexity: **O(1)**
